# README

## Beschreibung des Repositories

Dieses Repository enthält den gesamten Quellcode für die Studienarbeit zur Integration von Verkehrsdaten in Graphdatenbanken unter Verwendung von Neo4j. Der Code deckt alle Schritte von der Datenbeschaffung und -aufbereitung bis hin zur Modellierung und Analyse der Verkehrsdaten in Neo4j ab.

### Inhalt des Repositories

1. **Datenbeschaffung:**
   - Tabelle mit Links zu den Quellen der Verkehrsdaten

2. **Datenaufbereitung:**
   - Python-Skripte zur Formatierung und Bereinigung der heruntergeladenen Daten
   - Routinen zur Qualitätsprüfung und Dopplungsprüfung

3. **Modellierung in Neo4j:**
   - Cypher-Skripte zur Erstellung und Befüllung der Graphdatenbank mit den aufbereiteten Verkehrsdaten

### Voraussetzungen

- Python 3.x
- Neo4j
